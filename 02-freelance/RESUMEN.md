# RESUMEN — 02 Freelance (modo caveman)

Carpeta = textos listos para pegar en plataformas freelance (Fiverr, Workana, Freelancer, Toptal). Bios, gig, requisitos, miniaturas.

---

**perfil-freelancer.txt** — Perfil largo en español. Lista de servicios (web/móvil/landing/dashboards + IA: LLM, chatbots, RAG, automatización, auditoría costos), stack, experiencia producción (Smart Compliance, Analytiko, Altyro), cómo trabaja, diferencial ("ingeniero no prompt-engineer"). GMT-4, primera charla gratis.

**perfil-freelancer-600-en.txt** — Versión corta en inglés (~bio de 600 caracteres). Pitch denso: producción desde 2019, stack, logros (antifraude bancos, compliance Chile, 500 landings SEO con Claude, agente Twitch Llama 2). Para campos con límite de caracteres.

**workana-bio.txt** — Bio para Workana (español). Igual enfoque + lista de servicios CON tiempos reales de entrega (landing 2–4 días, chatbot RAG 3–5 días, integración LLM 2–5 días, app web MVP 2–4 sem, móvil 3–5 sem, auditoría costos 1–2 días).

**fiverr-gig-description.txt** — Descripción de gig Fiverr (inglés). Hook "idea → producto en días". Qué construye (chatbots, apps, SaaS MVP, automatizaciones, RAG, landings). Por qué él: entrega 3–14 días, costo predecible, código limpio. "First message is free."

**fiverr-buyer-requirements.txt** — 9 preguntas para el comprador antes de empezar (descripción, features must/nice, assets/API keys, plataforma, volumen usuarios/queries, docs para RAG, stack/hosting, deadline/presupuesto, comunicación). Sirve para cotizar bien y dimensionar costo tokens.

**reply-toptal.txt** — Respuesta corta declinando trabajo de traducción manual (él construye software, no traduce a mano). Ofrece referir a un traductor. Ejemplo de decir "no" con criterio.

**fiverr-thumbs/** — Miniaturas/imágenes del gig: `profile.png` + `thumb1/2/3.png` (artes) y `thumb1/2/3.html` (fuentes HTML que generan las miniaturas).

---

Nota: 4 variantes de bio (es largo, en corto, Workana, Fiverr). Mismo núcleo, distinto idioma/longitud/plataforma. Elegir según dónde se publica.
