Emprende / TeResuelve — app que conecta usuarios con negocios y servicios

Caso real · Por David Valor · AI Engineer en Caracas

Plataforma que conecta usuarios finales con negocios y servicios. La construí como un monorepo de 4 aplicaciones: app móvil (usuario), dashboard de administración, API REST central y landing de marketing.

Lo interesante: el backend

El API es un monolito modular en TypeScript que corre como Express en local y como AWS Lambda (serverless) en producción —el mismo código, dos modos—. Incluye autenticación JWT, registro con OTP por correo y comunicación en tiempo real por WebSocket.

Arquitectura

En producción todo vive serverless en AWS (us‑east‑1, dentro del free tier) conectado a MongoDB Atlas; los fronts se sirven desde S3. Costo de infraestructura cercano a cero, escalable cuando crezca.
