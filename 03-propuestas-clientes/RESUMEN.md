# RESUMEN — 03 Propuestas a clientes (modo caveman)

Carpeta = respuestas/propuestas a proyectos concretos. Plantillas adaptables a ofertas parecidas.

---

**propuesta-banca-chatbot.txt** — Chatbot bancario con LLM. Apoya en su paso por Analytiko (antifraude bancos). Cubre: NLP sobre consultas banca, RAG sobre docs internos, capa de orquestación (el LLM NUNCA toca la API core), OTP/MFA, escalado a humano, control costos (routing barato/caro, cacheo FAQs), logging/auditoría/anti prompt-injection. Cierra pidiendo llamada 30 min para cotizar.

**propuesta-ia-movil-robotica.txt** — Proyecto IA + móvil + robótica. Ofrece: features IA (LLM, RAG, agentes tool-calling, control costos), apps móviles iOS/Android (MVP 3–5 sem), capa software de robótica (control/orquestación hardware↔IA, visión, sensores). Honesto: firmware/electrónica → coordina con especialista hardware.

**propuesta-retirement-calc.txt** — Mantenimiento largo plazo de plataforma Next.js + TypeScript (dominio: cálculo de jubilación/seguros). Vende disciplina de mantenimiento: lee código antes de tocar, PRs pequeños, tests del bug, rigor numérico (redondeo/compounding/precisión). **Tarifa: 40 USD/h**, 10–20 h/sem, GMT-4. Busca relación de años, no contrato único.

**propuesta-sellside.txt** — Unirse a codebase en producción (SellSide, real estate). Responde 4 requisitos: (1) data pipeline/scraping (Analytiko ingestas + pipeline 500 landings Claude), (2) Supabase/PL-pgSQL (Smart Compliance Postgres/RDS, RLS, edge functions), (3) real estate = sin experiencia directa, honesto + transferible, (4) **tarifa 45 USD/h**, 15–20 h/sem (hasta 25). Async-first, overlap US.

**email-translation-app.txt** — Email de seguimiento: app de traducción de documentos EN→ES con LLM. Plantea build (upload→traduce→export, glosario/contexto para términos consistentes, costo/latencia primero). 5 preguntas de scope (formatos/volumen, fidelidad layout, plataforma, terminología, deadline/presupuesto). Demo cada viernes.

---

Patrón común en todas: empieza por dónde encaja → qué aporta (con experiencia real) → cómo trabaja (scope + costo antes de codear, demo semanal, handoff) → diferencial "ingeniero no prompt-engineer" → pide llamada para cerrar precio.
Tarifas de referencia: **40 USD/h** (mantenimiento) · **45 USD/h** (engagements más largos).
